# About
Program pemecah masalah _cryptarithmetic_ menggunakan algoritma _brute force_.

## Requirement
Memiliki Python3

## Cara Menggunakan
1. Buka terminal.
2. Buka path folder src dan jalankan `python3 cryptarithmetic.py` atau `./cryptaritmetic.py`
```sh
$ cd src
$ python3 cryptarithmetic.py
```
```sh
$ cd src
$ ./cryptarithmetic.py
```
